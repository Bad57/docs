# Default Particles

Default nanos world Particles and it's Parameters.

import * as BasicType from '/docs/components/_common-types.mdx';
import * as StructType from '/docs/components/_struct-types.mdx';

:::tip

**Note**: All Particles in this page are already included in the base game in the [Default nanos world Asset Pack](/docs/assets-modding/default-asset-pack/default-asset-pack), you can reference them like that: **`nanos-world::P_Beam`.**

:::

## Template Particles

Bare minimum particles.

### `P_Beam`

![](/img/docs/default-particles-beam.jpg)

This is a Niagara particle which joins two points with a Beam. Cool for creating Physics Guns or Laser effects.

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **`BeamColor`** | <StructType.Color /> | `Color(0, 0, 10, 1)` | Color of the Beam - multiply it to make it glow |
| **`BeamEnd`** | <StructType.Vector /> | `Vector(0, 0, 0)` | World position of the second joint |
| **`BeamWidth`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `5.0` | Beam thickness |
| **`JitterAmount`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `0.1` | Amount of jitterness |

### `P_Ribbon`

![](/img/docs/default-particles-ribbon.jpg)

This is a Niagara particle which makes a "trail" effect when moves.

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **`Color`** | <StructType.Color /> | `Color.WHITE` | Color - multiply it to make it glow |
| **`SpawnRate`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `100.0` | Rate of spawn the "ribbons" |
| **`Mass`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `10.0` | Mass of the "ribbons" |
| **`LifeTime`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `5.0` | How much time each "ribbon" will last |
| **`RibbonWidth`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `10.0` | The thickness of the Ribbon |

### `P_DirectionalBurst`

![](/img/docs/default-particles-directional-burst.jpg)

This is a Niagara particle which spawns a bunch of particles in a direction.

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **`Color`** | <StructType.Color /> | `Color.WHITE` | Color - multiply it to make it glow |
| **`SpawnCount`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `25.0` | Amount of Particles to spawn |
| **`VelocityStrengthMax`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `350.0` | Max velocity of the Particles |
| **`VelocityStrengthMin`** | [Float](/docs/scripting-reference/classes/particle#setparameterfloat) | `125.0` | Min velocity of the Particles |

### `P_Fountain`

![](/img/docs/default-particles-fountain.jpg)

### `P_HangingParticulates`

![](/img/docs/default-particles-hanging.jpg)

This is a Niagara particle which makes a 3-dimensional "hanging particles" effect.

| Parameter | Type | Default | Description |
